# API Endpoints

## General

```text
GET /
```

## Keys

```text
POST /messages/send
POST /messages/send/<key_num>
POST /messages/send/<key_num>/<executable_type>
```

## Rotary Encoder

```text
POST /encoder/send
POST /encoder/send/<encoder_action>
```

## OLED

```text
POST /oled/send
POST /oled/send/default
```
